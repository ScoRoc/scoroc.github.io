Legend of Tiny, Pixelated, 1980's Man

Legend of Tiny, Pixelated, 1980's Man is an HTML 5 canvas game that has 10 different enemies, including a 3 stage boss, each with their own custom build AI. The player can lose life upon being hit and gain life upon collecting a heart. In addition to the entire game, the custom backtrack was created by Scott as well.

Technologies used:

HTML | CSS | JavaScript | HTML5 Canvas | Bootstrap

SnapCap

This feature rich app offers users a fun way to upload pictures, caption other users’ pictures, and upvote the funniest pictures and captions. Newly uploaded pictures and captions are displayed in real time to provide a fluid social media experience. Dynamically rendered Upvotes and Followers allow users to see current top rated pictures and captions as well as see all of the people they are following.

Vue | Vuex | Python | Django | Materialize | CSS Grid | Axios
